Guide du joueur

Bienvenue dans
Ecologic

Ecologic est une simulation d'écosystème. Vous gérez un environnement naturel composé de trois espèces interdépendantes. Chaque année qui passe, les loups chassent les cerfs, les cerfs broutent l'herbe, et l'herbe repousse naturellement.

Votre rôle : observer cet équilibre — et intervenir si nécessaire.

🌿 Herbe
←
🦌 Cerfs
←
🐺 Loups
🐺 Les Loups Chassent les cerfs chaque année pour survivre. Trop de loups = famine.
🦌 Les Cerfs Broutent l'herbe pour se nourrir. Trop de cerfs = prairie dévastée.
🌿 L'Herbe Repousse naturellement chaque année. La base de tout l'écosystème.
⚖️

L'équilibre fragile

La nature est cruelle, mais logique. Trois scénarios peuvent briser l'équilibre :

  • Trop de loups — ils déciment les cerfs. Sans cerfs, ils meurent de faim.
  • Trop de cerfs — ils épuisent toute l'herbe et périssent à leur tour.
  • Plus assez d'herbe — toute la chaîne alimentaire s'effondre.
🎛️

Comment intervenir

  • Modifier les paramètres — ajustez la fréquence de reproduction, l'agressivité des loups, la vitesse de repousse de l'herbe.
  • Ajouter des individus — introduisez directement loups, cerfs ou herbe en cours de partie.
  • Avancer le temps — simulez plusieurs années d'un coup avec le mode accéléré.
🌦️

Les événements météo

Des événements aléatoires viennent bousculer l'équilibre au fil des années. À vous de vous adapter.

  • ☀️ Sécheresse — l'herbe se raréfie, les cerfs souffrent.
  • ❄️ Hiver rigoureux — toutes les espèces sont affectées.
  • 🔥 Incendie — l'herbe est dévastée brutalement.
  • 🌧️ Pluies abondantes — l'herbe explose, les cerfs prolifèrent.
  • 🌸 Printemps précoce — tout l'écosystème s'emballe.
🎯

L'objectif

La partie se termine quand une espèce disparaît complètement. Plus vous tenez longtemps, mieux c'est.

Et si vous vous sentez d'attaque, tentez de débloquer les 9 succès cachés… Chacun récompense un exploit particulier dans l'écosystème.
