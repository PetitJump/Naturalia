# Naturalia

Fonctionnement du projet:

terminal.py est un fichier qui fonctionne sur terminal car le Flask avec app.py est en développement.

Lorsque Flask sera terminé:
app.py est le fichier où il faut executer le programme. 
les fonctions principales se trouvent dans algo.py et celles-ci sont stockées dans une fontction update qui va être utilisé dans le app.py.
Les données brut des animaux sont stockées dans un fichier json tandis que les données des utilisateurs qui vont se connecter vont être stockés dans une base de donnée en SQL.
